# Small-CA - A Small Certificate Authority

## Introduction 

**Small-CA** is a set of bash scripts that uses OpenSSL to create a small Certificate Authority for small intranets.

## Dependencies

* Bash 4.4+;
* OpenSSL 1.1.0+;
